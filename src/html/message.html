Discontinuing support

Support for the 'Save to the Wayback Machine' browser extension will be discontinued on January 1st 2023.

Google Chrome & Microsoft Edge

From January 2023, the browser extension will no longer run on Google Chrome or Microsoft Edge as both web browsers are dropping support Manifest V2 powered extensions.

Firefox

The browser extension will continue to work in Firefox for the foreseeable future but will not receive updates.

Need a replacement?

We recommend using the Internet Archive's official Wayback Machine browser extension as it includes all the same features, and is available for Google Chrome, Firefox and Microsoft Edge.
